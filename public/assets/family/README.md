# Family photos

Drop Crissy & Connor's photos into this folder. Each one appears automatically
on the **Our Story** page once the file is here — no code changes needed. Use
`.jpg`, `.jpeg`, `.png`, or `.webp` (phone **.heic** files won't display in
browsers, so export/convert them to **.jpg** first).

Use these exact base names (extension can be any of the above):

| File name (base)        | Where it appears on the page          |
| ----------------------- | ------------------------------------- |
| `crissy-before`         | Large memorial portrait — "Crissy, before Connor" |
| `crissy-connor-birth`   | "The day Connor was born"             |
| `crissy-connor-baby`    | "A week or so later, home together"   |
| `crissy-connor-2021-a`  | "Crissy and Connor, 2021"             |
| `crissy-connor-2021-b`  | "Mother and son"                      |

Example: `crissy-before.jpg`, `crissy-connor-birth.jpg`, etc.

## Easiest way to add them (from a phone or computer)

1. Open the pull request for this branch on GitHub.
2. Browse to `public/assets/family/` and choose **Add file → Upload files**.
3. Rename each photo to the matching base name above and upload.
4. Commit to the same branch. The site will show them on the next build.

Captions live in `app/story/page.js` if you'd like to reword any of them.
